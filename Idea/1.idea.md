## 快捷键  
1.全文搜索： Ctrl + Shift + R  
2.实现类： Ctrl + Alt + B  
3.快速返回上次查看代码的位置： ctrl + Alt + <-(| ->)  
**注意请修改系统：显示 -> 高级设置 -> 英特尔e核芯显卡控制面板 -> 图形属性 -> 选项和支持 -> 禁用快捷键**  
4.关闭标签： Ctrl + F4  
5.显示方法列表： ALT + 7  
6.去掉多余引用： CTRL + ALT + O  
7.查看变量（方法登）哪里被使用： ALT + F&  
8.大小写转换： CTRL + SHIFT + U
