# TCP/IP



## 1. TCP/IP 基础知识



**TCP（Transmission Control Protocol）和 IP（Internet Proto-col）**是互联网的众多通信协议中最为著名的。

### 1.1 TCP/IP 标准化

TCP/IP 并不是我们特指的 TCP 协议，IP 协议，而是泛指一个协议群，有时也称 TCP/IP 为 **网际协议族**

**TCP/IP 协议群包括以下内容:**

- **应用协议**： HTTP, SMTP, FTP, TELNET, SNMP
- **传输协议**：TCP, UDP
- **网际协议**：IP, ICMP, ARP
- **路由控制协议**：RIP, OSPF, BGP
