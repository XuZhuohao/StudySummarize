### 1.2 liunx 学习阶段

- 抛弃旧的思维习惯，熟练使用 Linux 命令行
  - 《鸟哥的Liunx私房菜》
  - 《Liunx系统管理技术手册》
- 通过系统调用或者 glibc，学会自己进行程序设计
  - 《UNIX环境高级编程》
- 了解 Linux 内核机制，反复研习重点突破
  - 《深入理解Liunx内核》
- 阅读 Linux 内核代码，聚焦核心逻辑和场景
  - 《Liunx内核源代码情景分析》
- 实验定制化 Linux 组件，已经没人能阻挡你成为内核开发工程师了
- 面向真实场景的开发，实践没有终点
